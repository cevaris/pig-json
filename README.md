pig-json
========

JSON parser/loader for Pig &lt; 0.10.0

Sourced from here
[https://github.com/mmay/PigJsonLoader](https://github.com/mmay/PigJsonLoader)

## Quickstart

1. Get the code: `git clone https://github.com/cevaris/pig-json.git`
1. Build the jar: `mvn package`
